# Backend Setup

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Copy `.env.example` to `.env` and fill in the required values.
3. Start the development server:
   ```bash
   node index.js
   ```

## Tech Stack

- Node.js
- Express
- MongoDB (Mongoose)
- Cloudinary
- Google APIs (OAuth, Gmail)
- Nodemailer
- Multer
- dotenv
- cors
